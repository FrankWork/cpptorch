## git-lfs

```bash
wget https://github.com/git-lfs/git-lfs/releases/download/v2.6.0/git-lfs-linux-amd64-v2.6.0.tar.gz
mkdir git-lfs-linux
tar zxvf git-lfs-linux-amd64-v2.6.0.tar.gz -C git-lfs-linux/
cd git-lfs-linux
sudo bash install.sh
```

```bash
git clone https://github.com/baidu/lac.git
```
